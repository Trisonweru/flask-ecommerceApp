This project implements an e-commerce website using python Flask Framework and an sqllite database. 
It also integrates a payment platform using stripe to enable user to pay via card.

How to run Online Store
1) Install the requirements of this project, check requirements.txt and from the virtual environment of this project.
   Run pip3 install -r requirements.txt to make the installations.
2) After installation run the python file in ru.py to get started.
NB: The data used for testing is in the SQL database in the shop folder shipped with this project, check it there.
You could install SQL DB browser to your machine and check the data.
